# Task 1 — Responsive Landing Page

A clean, responsive landing page built with HTML5 and CSS3 as part of the **ElevateLabs Web Development Internship**.

---

## 📌 Objective

Build a responsive landing page with a header, hero section, and footer using only HTML and CSS.

---

## 🗂️ Files

| File | Description |
|------|-------------|
| `index.html` | Main HTML structure of the page |
| `style.css` | All styling, layout, and responsive rules |

---

## 🌐 Sections

- **Header** — Fixed navbar with logo, navigation links, and a "Get Started" CTA button
- **Hero** — Headline, subtext, call-to-action buttons, and floating stat cards
- **Features** — 4-column grid showcasing key product highlights
- **Footer** — Brand info, link columns, and social media icons

---

## 💡 Concepts Used

- **HTML5** semantic elements (`<header>`, `<main>`, `<section>`, `<footer>`, `<nav>`)
- **CSS Flexbox** — used for header layout, nav alignment, and footer brand section
- **CSS Grid** — used for hero two-column layout, features grid, and footer columns
- **Media Queries** — responsive breakpoints at `768px` and `400px`
  - Nav collapses into a hamburger menu on mobile
  - Hero stacks to a single column
  - Features and footer reflow to single column
- **CSS Variables** — consistent theming via `:root` custom properties
- **CSS Animations** — floating card effect using `@keyframes`
- **JavaScript** — hamburger toggle and sticky header scroll effect

---

## 📱 Responsive Breakpoints

| Breakpoint | Behaviour |
|------------|-----------|
| `> 960px` | Full desktop layout |
| `≤ 960px` | Features become 2-column, footer adjusts |
| `≤ 768px` | Single column, hamburger nav, hero visual hidden |
| `≤ 400px` | Buttons stack vertically |

---

## 🚀 How to Run

1. Clone or download this repository
2. Open the folder in **VS Code**
3. Install the **Live Server** extension
4. Right-click `index.html` → **Open with Live Server**
5. Resize the browser window to test responsiveness

---

## 🛠️ Tools Used

- VS Code
- Chrome Browser
- Live Server extension
- Google Fonts (Syne + DM Sans)
